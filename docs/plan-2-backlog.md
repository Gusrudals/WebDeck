# Plan 2 백로그 — 포맷 v1 알려진 갭

Plan 1 최종 리뷰(2026-07-02)에서 식별된, 머지는 막지 않지만 Plan 2(에디터 코어)에서 다뤄야 할 항목.

## 검증기

- **중첩 `.el` 미검사**: 가이드 규칙 13은 `.el` 중첩을 금지하지만 검증기는 슬라이드 직속 자식만 검사한다. Plan 2의 파서가 이 불변식에 의존하므로 재귀 검사(`el.querySelector('.el')`) 추가 필요
- **자기완결성 잔여 구멍**: `<style>` 내 `@import url(...)`, 인라인 style의 `url()`, `<iframe src>`, `<video src>`는 현재 통과함. 검증 범위 확대 검토
- **`parseInlineStyle`은 검증 전용**: `;` 단순 분리라 값에 세미콜론이 포함되면(`url(data:image/svg+xml;base64,...)`) 값이 잘린다. 검증 용도로는 무해하지만 **Plan 2 왕복 직렬화에 재사용 금지** — 파서는 별도 구현 필요

## CLI

- **`validateWebdeck` 예외 시 종료 코드**: 검증기가 던지면 uncaught로 종료 코드 1(포맷 오류로 오독됨)이 된다. AI 생성 루프가 문서 수정을 무한 반복할 수 있으므로 try/catch로 감싸 2로 종료하도록 수정 (4줄)
- **cli.test.mjs tmpdir 미정리**: 손대는 김에 `after()` 훅 추가
